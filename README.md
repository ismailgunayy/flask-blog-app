# Flask Blog App

### Setting up the environment

```
pip install Flask
pip install Flask-WTF
pip install passlib
pip install wtforms
```
In the `app` directory,
```
set FLASK_APP=app.py
flask run
```

Go to `localhost:5000/`

Create an account and write some articles to see how the system works.
